# binaryPetsClaude

Claude Code plugins marketplace by binaryPets.

## Installation

```bash
/plugin marketplace add anouar1991/binaryPetsClaude
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [rules-learning-pipeline](./plugins/rules-learning-pipeline) | Automated learning pipeline for optimizing CLAUDE.md rules |

## Usage

After adding the marketplace, install plugins with:

```bash
/plugin install <plugin-name>@binaryPetsClaude
```
